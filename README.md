# Weekly for Evernote

This is a sample python script to generate my weekly TODO list, I will use it
with Evernote.

## Requirements

1. Python 3+

## Make

Generate this week TODO list.

```console
make or make build
```

Check the lint with flake8.

```console
make check
```

Clean artifact.

```console
make clean
```
